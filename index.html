About the Project

CoFu is a mobile puzzle game where players combine same-colored tiles in a match-3 style gameplay. Built with performance optimization and engaging mechanics for mobile platforms.

The game features smooth physics-based interactions, intuitive controls, and a scalable architecture using Scriptable Objects for data management.

Key Features

  • Smooth tile matching mechanics with satisfying feedback
  • Physics-based tile interactions for dynamic gameplay
  • Event-driven UI system for responsive interface
  • Scalable level progression system
  • Optimized for mobile performance

Technologies & Architecture

Unity 3D Physics system, particle effects
C# Programming Observer pattern, OOP principles
Scriptable Objects Data-driven architecture
Event System Decoupled UI communication
